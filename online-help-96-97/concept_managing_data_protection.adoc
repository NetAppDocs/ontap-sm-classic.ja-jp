= データ保護の管理
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager を使用して、ミラー関係、バックアップ関係、ミラーとバックアップ関係を作成および管理し、データを保護することができます。Snapshot ポリシーとスケジュールを作成および管理することもできます。
