= DNS / DDNS サービスウィンドウ
:icons: font
:imagesdir: ../media/


[role="lead"]
DNS / DDNS サービスウィンドウを使用すると、システムの現在の DNS および DDNS の設定を確認して編集できます。



== コマンドボタン

* * 編集 * 。
+
DNS / DDNS 設定の編集ダイアログボックスを開きます。このダイアログボックスを使用して、 DNS または DDNS の詳細を追加または変更できます。DDNS を有効または無効にすることもできます。

* * 更新 *
+
ウィンドウ内の情報を更新します。
