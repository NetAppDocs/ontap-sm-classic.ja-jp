= クラスタの更新
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager を使用して、ハイアベイラビリティ（ HA ）ペアのクラスタまたは個々のノードを更新できます。MetroCluster 構成のクラスタも更新できます。
