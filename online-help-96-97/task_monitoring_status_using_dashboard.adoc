= ダッシュボードを使用したクラスタの監視
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager のダッシュボードを使用すると、クラスタの健常性とパフォーマンスを監視できます。また、ハードウェアやストレージ構成の問題を特定することもできます。

.手順
. ダッシュボード * タブをクリックすると、ヘルス / パフォーマンスダッシュボードパネルが表示されます。
