= イニシエータホスト
:icons: font
:imagesdir: ../media/


[role="lead"]
イニシエータホストは自身にマッピングされた LUN にアクセスできます。ストレージシステム上の LUN を igroup にマッピングすると、グループ内のすべてのイニシエータからその LUN へのアクセスを許可することになります。LUN にマッピングされている igroup のメンバーでないホストは、 LUN にアクセスできません。
