= System Manager を使用してクラスタを設定する
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager を使用してクラスタを設定するには、一定の前提条件を満たしている必要があります。

* クラスタを作成しておく必要があります。
* クラスタが設定されていない状態である必要があります。
