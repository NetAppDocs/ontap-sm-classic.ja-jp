= SVM の起動
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager を使用して Storage Virtual Machine （ SVM ）を起動することで、 SVM からデータアクセスを提供できます。

.手順
. [ * ストレージ * > * SVMs * ] をクリックします。
. 開始する SVM を選択し、 * Start * をクリックします。


SVM によるクライアントへのデータの提供が開始されます。
