= NFS で Kerberos を使用してセキュリティを強化する
:icons: font
:imagesdir: ../media/


[role="lead"]
Kerberos を使用して SVM と NFS クライアント間の認証を強化すると、セキュアな NFS 通信を実現することができます。NFS に Kerberos を設定すると、 NFS クライアントとストレージシステム間の通信の整合性とセキュリティが向上します。
