= Snapshot 設定
:icons: font
:imagesdir: ../media/


[role="lead"]
既存の Snapshot ポリシーにスケジュールを設定することで、 Snapshot コピーを設定できます。ONTAP 9.4 以降では、 FlexVol ボリュームの Snapshot コピーを最大 1024 個まで保持できます。
