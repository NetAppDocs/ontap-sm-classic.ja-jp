= システムヘルスアラートの抑制
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager を使用して、手動の操作を必要としないシステムヘルスアラートを抑制できます。

.手順
. [* Events & Jobs*>* System Alerts*] をクリックします。
. [ * システム・アラート * ] ウィンドウで、サブシステム名の横にある矢印アイコンをクリックします。
. 抑制するアラートを選択し、 * 抑制 * をクリックします。
. 名前を入力し、 * 抑制 * をクリックします。
