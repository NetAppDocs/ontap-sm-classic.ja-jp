= ポートセットを削除します
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager を使用して、不要になったポートセットを削除できます。

.手順
. [ * ストレージ * > * LUNs * ] をクリックします。
. [ * ポートセット * ] タブで、 1 つまたは複数のポートセットを選択し、 [ * 削除 * ] をクリックします。
. 削除を確認するには、 * Delete * をクリックします。
