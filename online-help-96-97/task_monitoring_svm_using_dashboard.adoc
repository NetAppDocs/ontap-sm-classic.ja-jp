= SVM の監視
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager のダッシュボードを使用すると、 Storage Virtual Machine （ SVM ）の健常性とパフォーマンスを監視できます。

.手順
. [ * ストレージ * > * SVMs * ] をクリックします。
. 監視する SVM を選択します。
. ダッシュボードパネル内の詳細情報を確認します。
