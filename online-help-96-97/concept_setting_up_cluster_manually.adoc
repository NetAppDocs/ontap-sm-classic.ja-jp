= クラスタを手動でセットアップする
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager を使用してクラスタを手動でセットアップできます。クラスタを作成し、ノード管理ネットワークとクラスタ管理ネットワークのセットアップ、およびイベント通知のセットアップを行います。
