= CIFS の Kerberos 認証
:icons: font
:imagesdir: ../media/


[role="lead"]
Kerberos 認証では、クライアントは CIFS サーバへの接続時に使用可能な最も高いセキュリティレベルをネゴシエートします。ただし、クライアントが Kerberos 認証を使用できない場合は、 Microsoft NTLM または NTLM V2 を使用して CIFS サーバでの認証が行われます。
