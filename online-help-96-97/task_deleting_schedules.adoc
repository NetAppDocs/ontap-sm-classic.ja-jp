= スケジュールを削除します
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager を使用して、特定のストレージ管理タスクを実行するスケジュールを削除できます。

.手順
. [* 保護 *>* スケジュール *] をクリックします。
. 削除するスケジュールを選択し、 * Delete * をクリックします。
. 確認のチェックボックスをオンにし、 * 削除 * をクリックします。
