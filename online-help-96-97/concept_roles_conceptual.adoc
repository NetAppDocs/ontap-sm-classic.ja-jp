= ロール
:icons: font
:imagesdir: ../media/


[role="lead"]
アクセス制御ロールを使用して、ユーザのシステムへのアクセスのレベルを制御できます。事前定義されたロールを使用するほか、新しいアクセス制御ロールを作成したり、変更したり、削除したりできます。また、ロールのユーザに対してアカウントの制限も指定できます。
