= ピア関係の設定（ ONTAP 9.2 以前）
:icons: font
:imagesdir: ../media/


[role="lead"]
ピア関係で定義されるネットワーク接続により、クラスタ間および SVM 間でデータをセキュアにやり取りすることができます。SVM ピア関係を作成する前に、クラスタピア関係を作成する必要があります。
